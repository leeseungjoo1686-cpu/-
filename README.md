# -급할 때 바로 쓰는 AI 파일 종합 분석기입니다!

1. 보낸 링크에서 'app.py' 파일 딱 하나만 다운로드합니다.
2. 터미널(또는 명령 프롬프트)을 열고 파일이 있는 폴더로 이동합니다.
3. 아래 명령어를 치면 필요한 프로그램 설치부터 실행까지 한 번에 완료됩니다:

   streamlit run app.py
   @echo off
echo ===================================================
echo  ⚡ AI FILE ANALYZER 클라우드 서버 접속 중...
echo ===================================================

:: 내 컴퓨터에서 복잡하게 실행하지 않고, 만들어진 웹사이트 주소로 바로 보냅니다.
🚀 잠시 후 AI 분석기 웹사이트가 브라우저에서 열립니다!
start https://hb5gbjgc6vkkud5fub5mgb.streamlit.app/

exit
팀원별 역할
1.이승주: 팀원들이 제작한 코드를 받아서 깃허브에 머지후 사이트 제작 및 수정
2.전은서: 기초적인 코드 제작 및 여러가지 수정사항 제시
2.최윤서: 발표자료 제작
